# தமிழர் தெலுங்கர் திராவிட முன்னேற்ற ஒற்றுமை கழகம் (MJK India)

அரசியல் பாதை, புதிய சிந்தனை, புதிய முயற்சிகள்.

## அறிமுகம் (Introduction)

தமிழர் தெலுங்கர் திராவிட முன்னேற்ற ஒற்றுமை கழகம் (MJK India) என்பது பிற்படுத்தப்பட்டோர், மிகவும் பிற்படுத்தப்பட்டோர் மற்றும் தாழ்த்தப்பட்டோர் ஆகிய பிரிவினரை கல்வி, சமூகம், பொருளாதாரம், அரசியல் ரீதியாக முன்னேற்றுவதற்காக உருவாக்கப்பட்ட ஒரு இயக்கமாகும். இந்த இயக்கத்தை **மக்கள் தலைவர் திரு. மனோகரன் கே.ஏ** அவர்கள் நிறுவி வழிநடத்தி வருகிறார்.

## முக்கிய அம்சங்கள் (Key Features)

*   **கொள்கைகள் & நோக்கம்**: சமுதாய முன்னேற்றம் மற்றும் இயற்கை வளங்களை பாதுகாப்பதற்கான செயல் திட்டங்கள்.
*   **நிர்வாகிகள்**: கழகத்தின் முக்கிய நிர்வாகிகள் மற்றும் அவர்களின் செயல்பாடுகள்.
*   **ஒளிப்படத் தொகுப்பு (Gallery)**: கட்சியின் பல்வேறு நிகழ்வுகள் மற்றும் பொதுக்கூட்டங்களின் புகைப்படங்கள்.
*   **நிகழ்வுகள்**: தேர்தல் பிரச்சாரம், வேட்பு மனு தாக்கல் மற்றும் மக்கள் சந்திப்பு நிகழ்ச்சிகள்.
*   **சின்னம் (Symbol)**: கழகத்தின் வெற்றி சின்னம் - **வீணை**.

## தொழில்நுட்பங்கள் (Technologies Used)

이 இணையதளம் பின்வரும் தொழில்நுட்பங்களைக் கொண்டு உருவாக்கப்பட்டுள்ளது:

*   **HTML5**: இணையதளத்தின் கட்டமைப்பு.
*   **CSS3**: நவீன மற்றும் நேர்த்தியான வடிவமைப்பு (Vanilla CSS).
*   **JavaScript**: மொபைல் மெனு மற்றும் போட்டோ கேலரியின் (Lightbox) செயல்பாடுகள்.

## நிறுவுதல் மற்றும் பயன்படுத்துதல் (Installation and Usage)

இந்த இணையதளத்தை உங்கள் கணினியில் இயக்க:

1.  இந்தத் திட்டத்தை (Repository) பதிவிறக்கம் செய்ய அல்லது குளோன் (Clone) செய்யவும்.
2.  `index.html` கோப்பை எந்தவொரு இணைய உலாவியிலும் (Browser) திறக்கவும்.

## கட்டமைப்பு (Project Structure)

```text
├── index.html       # இணையதளத்தின் முகப்புப் பக்கம்
├── index.css        # வடிவமைப்புத் தாள் (Styles)
└── asset/           # புகைப்படங்கள் மற்றும் வீடியோக்கள்
```
